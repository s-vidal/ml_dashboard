# ml_dashboard

### steps to run code

**In Terminal:**

> 1.  mkdir dashboard
> 2.  cd dashboard
> 3.  git clone "https://github.com/s-vidal/ml_dashboard.git"
> 4.  cd ml_dashboard
> 5.  cd counterfactuals
> 6.  npm install
> 7.  npm start
> 8.  open browser on localhost://3000

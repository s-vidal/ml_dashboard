# ml_dashboard

### steps to run code

_In Terminal:_

1.  mkdir dashboard
2.  cd dashboard
3.  git clone "https://github.com/s-vidal/ml_dashboard.git"
4.  cd counterfactuals
5.  npm install
6.  npm start
7.  open browser on localhost://3000
